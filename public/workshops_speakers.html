Workshops

Linux (pdf)

An introduction to debian linux and the Bourne Again SHell (bash). Learn how to navigate the linux terminal, manipulate folders and files, compile and run your code, and become familiar with common tools.

  • Git (pdf)

    The world's most popular version control system. Learn how git functions and explore the features that streamline the software development process.

  • Python (pdf)

    Python is an interpreted programming language that lets you work quickly and integrate systems effectively. In this workshop, we explore the interpreted language, significant whitespace, and several common methods available through the Python API.

  • Verilog Synthesis (pdf)

    Verilog is a popular Hardware Description Language used in FPGAs and Register-Transfer Level design. In this workshop, we do a quick overview of a sample design and implement it onto the Spartan 3.

  • Debugging (pdf)

    Identifying and solving problems in code efficiently is a valuable skill for any programmer. In this workshop, we explore tools and techniques for smashing these pesky bugs.

  • Soldering (pdf)

    A well soldered electical connection conducts electricity better and is more resistant to changes in the environment. Learn how to safely and effictively solder wires, PCBs, and other common electrical components.

  • Arduino (pdf)

    Arduino is a great open-source prototyping board for exploring electronics, prototyping, and making fun widgets. Learn how to get started with your arduino!

  • Arduino (pdf)

    Vim is an advanced text editor that seeks to provide the power of the de-facto Unix editor 'Vi', with a more complete feature set.
